## 📝 Lista de Exercícios: Lógica de Programação

### Tópicos: if/else, while, for e Listas.

### Parte 1: Estruturas Condicionais (if)

- Verificação de Acesso: Crie um programa que peça a idade do usuário. Se for maior ou igual a 18, exiba "Acesso Permitido", caso contrário, "Acesso Negado".

- Status de Servidor: Escreva um script que receba um número (0 ou 1). Se for 1, imprima "Servidor Online". Se for 0, imprima "Servidor Offline".

- Par ou Ímpar: Receba um número inteiro e diga se ele é par ou ímpar (Dica: use o operador %).

- Calculadora de Desconto: Um curso de hardware custa R$ 500,00. Se o aluno pagar à vista, tem 10% de desconto. Se for parcelado, o preço é normal. Calcule o valor final.

- Classificação de IP: Peça o primeiro octeto de um IP. Se estiver entre 1 e 126, diga que é "Classe A". Se estiver entre 128 e 191, "Classe B". (Simplificado).

### Parte 2: Repetição com while

- Contagem Regressiva: Crie um programa que conte de 10 até 0 e depois imprima "Foguete lançado!".

- Validação de Senha: Crie um while que peça uma senha e só pare de pedir quando o usuário digitar "admin123".

- Somador Infinito: Peça números ao usuário e vá somando-os. O programa para quando o usuário digitar o número 0.

- Menu de Opções: Faça um menu que exiba: 1- Adicionar, 2- Remover, 3- Sair. O programa deve continuar rodando até que o usuário escolha a opção 3.

- Média de Notas: Peça notas de alunos uma por uma. Quando o usuário digitar -1, o programa para e mostra a média das notas digitadas.

### Parte 3: Repetição com for

- Tabuada Dinâmica: Peça um número ao usuário e exiba a tabuada dele de 1 a 10 usando for.

- Busca de Caractere: Peça uma frase e conte quantas vezes a letra "a" aparece nela.

- Números Pares: Use um for com range() para imprimir todos os números pares entre 1 e 50.

- Quadrado de Números: Imprima o quadrado de cada número de 1 a 10 (Ex: 1, 4, 9, 16...).

- Simulação de Ping: Crie uma lista com 4 IPs e use o for para imprimir "Pingando no IP: [endereço]" para cada um da lista.

### Parte 4: Listas (Arrays)

- Lista de Compras: Crie uma lista com 5 itens de hardware. Imprima apenas o terceiro item da lista.

- Atualização de Inventário: Dada a lista estoque = ["Mouse", "Monitor"], adicione "Teclado" ao final e substitua "Mouse" por "Mouse sem fio".

- Maior Valor: Crie uma lista com 5 números inteiros e utilize um laço para encontrar e imprimir o maior número dela.

- Remoção de Usuário: Crie uma lista de usuários logados. Peça o nome de um usuário e, se ele estiver na lista, remova-o.

- Lista Inversa: Crie uma lista com os nomes dos dias da semana e imprima-os na ordem inversa (do domingo para a segunda).
